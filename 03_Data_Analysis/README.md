# Phase 3 — Data Analysis

Data analysis is the process of inspecting, cleaning, transforming, and modeling data to discover insights and support decisions. This phase covers the full Python-based analytical stack plus BI and visualization tools.

---

## Learning Steps

### 1. NumPy — Numerical Computing
- Multidimensional arrays, vectorized operations, broadcasting
- Mathematical functions: linear algebra, statistics, Fourier transforms
- Foundation for Pandas, Scikit-learn, and PyTorch

### 2. Pandas — Data Manipulation
- Series and DataFrames for structured data
- Loading data: CSV, JSON, Excel, SQL, Parquet
- Cleaning: missing values, duplicates, type casting
- Transformations: groupby, merge, pivot, melt, reshape
- Time-series indexing and resampling

### 3. Statistics for Data Analysis
- Descriptive stats: mean, median, mode, variance, standard deviation
- Probability distributions: normal, binomial, Poisson
- Hypothesis testing: t-test, chi-squared, ANOVA
- Correlation, covariance, and feature relationships

### 4. Exploratory Data Analysis (EDA)
- Understanding distributions and outliers
- Feature correlation heatmaps
- Pair plots, box plots, violin plots
- Identifying data quality issues before modeling

### 5. Data Visualization
- Choosing the right chart for your data type
- Matplotlib: full control over figure-level plots
- Seaborn: statistical visualizations with minimal code
- Plotly: interactive charts for dashboards and web
- Best practices: labels, colors, accessibility, storytelling

### 6. Business Intelligence & Dashboard Tools
- Power BI: drag-and-drop dashboards for business reporting (DAX formulas)
- Tableau: advanced visual analytics and storytelling
- Grafana: real-time operational dashboards (connects to Phase 8 robot telemetry)
- Looker Studio: Google's free BI tool
- Apache Superset / Metabase: open-source BI platforms
