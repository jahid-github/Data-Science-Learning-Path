# Resources

This folder is the portfolio entry point for the learning materials already stored in the repository.

The actual study content still lives in the numbered phase folders at the repository root. Those folders are kept in place to avoid breaking links and to preserve the learning roadmap.

## How To Use The Existing Material

Use the numbered phase folders as:

- a study roadmap
- a topic index
- a quick reference library

Do not treat the PDF collection alone as portfolio proof.

Recruiters care much more about:

- runnable projects
- evaluation results
- engineering decisions
- demos
- reproducibility

## Rule For Future Updates

When adding a new PDF, cheat sheet, or article summary:

- place it in the correct phase folder
- add a short reason why it matters
- link it from the relevant phase `README`

When adding work that proves skill:

- put it under [../projects](../projects/)
- keep results, setup, and metrics close to the code
