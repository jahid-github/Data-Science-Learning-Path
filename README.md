## Machine Learning, Deep learning & Data Analysis

- **NumPy** (multidimensional array, numerical data)
- **Pandas** (data manipulation and analysis, Handles mixed types (floats, strings, dates), 2D DataFrames (Tabular))
- **PyTorch** (custom models, edge AI, real-time inference)
- **Scikit-learn** (regression, clustering, evaluation metrics)
- **TensorFlow** (deploying scalable models)
- **XGBoost** (tabular prediction (demand forecasting, business ML))
- **OpenCV** (preprocessing, segmentation, HSV masking, pipelines)
- **YOLO (You Only Look Once)** (object detection system that identifies and locates multiple objects in an image or video in a single pass through a neural network, prioritizing incredible speed for real-time applications.)
- **MediaPipe** is a modular, cross-platform framework developed by Google that allows you to build production-ready multimodal Machine Learning pipelines (video, audio, and sensor data) for real-time perception on mobile, web, and edge devices.
- **Detectron2** is a high-performance, modular library by Meta AI (FAIR) that provides state-of-the-art algorithms for object detection, instance segmentation, and panoptic segmentation using a PyTorch-based framework.
