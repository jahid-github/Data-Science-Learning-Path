## 4. Machine Learning

Machine Learning is a part of AI where systems learn from data and improve automatically without being explicitly programmed for every task.

### Main Types of Machine Learning

#### Supervised Learning

The model learns from labeled data.

Examples:

* Email spam classification
* House price prediction

#### Unsupervised Learning

The model learns patterns from unlabeled data.

Examples:

* Customer segmentation
* PCA for dimensionality reduction

#### Reinforcement Learning

An agent learns by interacting with an environment and receiving rewards or penalties.

Examples:

* AlphaGo
* Robotics navigation

### Additional Machine Learning Types

* Semi-supervised Learning
* Transfer Learning
* Deep Learning

### Machine Learning Applications

* Predictive analytics
* NLP
* Computer vision
* Recommendation systems
* Robotics
* Finance

**How do machines learn patterns from data and make predictions or decisions?**

## 5. Deep Learning

Deep Learning is a specialized part of Machine Learning based on neural networks with many layers.

It is especially powerful for large and complex data such as images, text, audio, and video.

### Key Characteristics

* Layered neural network structure
* Automatic feature extraction
* High computational requirement
* Works well with large datasets
* End-to-end learning

### Deep Learning Applications

* Image recognition
* NLP
* Autonomous vehicles
* Speech recognition
* Recommendation systems
* Gaming

### Main Deep Learning Frameworks

**TensorFlow**
Used for large-scale neural network training and production.

**Keras**
High-level API for quick and easy deep learning model building.

**PyTorch**
Flexible and widely used in research and modern deep learning development.
